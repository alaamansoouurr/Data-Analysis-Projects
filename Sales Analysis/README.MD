# Sales Analysis Dashboard
## Overview
Analysis Sales using Power Bi
## Tools Used
- Power Bi
- DAX
## Key Insights
- Total Revenue / Total Quantity / Total Profit / Total Discount
- Relation Between Customer & Revenue
- Relation Between Order & Revenue
- Total_Rev Per Segment
- Total_Rev Per Ship Mode
- Revenue Ranking Per Product Name
- Profit Ranking Per Product Name
- Revenue Ranking Per Customer Name
- Profit Ranking Per Customer Name
- MOM_Rev & Total_Rev Per Month
- MOM_Profit & Total_Rev Per Month
- Rev_LY & Total_Rev Per Year
- Profit_LY & Total_Profit Per Year
